## Success Criteria
- make -C build succeeds with 0 errors
- All tests pass

## Tools Available
- ctx7: query latest documentation (ctx7 library <name> <query>)
- make -C build: build the project
- git: commit changes

## Exit Conditions
- All Q1 ROADMAP items are checked off
- BUILD_FAILED after 3 attempts
